## `> SYSTEM.OVERVIEW`

**JARVIS Telemetry** is a cinema-grade macOS heads-up display that transforms your desktop into a real-time Iron ManвҖ“style reactor interface. It renders a full-screen arc reactor HUD at wallpaper level вҖ” sitting beneath all windows while streaming live Apple Silicon telemetry data from CPU, GPU, memory, and thermal sensors.

Every ring, tick mark, bezel panel, and data arc is drawn at **60fps** in a SwiftUI `Canvas` вҖ” no images, no WebKit, no Electron. Pure GPU-accelerated vector rendering.
